# Cookiecutter Haskell

### Features

- Uses Cabal and [hpack](https://github.com/sol/hpack).
- Configures [relude](https://hackage.haskell.org/package/relude) as the Prelude.
- Sets up Nix, [Flake Parts](https://flake.parts), and [git-hooks.nix](https://github.com/cachix/git-hooks.nix).
- Optional [REUSE](https://reuse.software/) preset.
